# Sathington's Dotfiles

Personal Linux configuration for my primary desktop.

## System

- Distribution: CachyOS
- Desktop: KDE Plasma 6 (Wayland)
- Terminal: Kitty
- Shell: Fish
- Theme: Catppuccin Mocha

## Included Configurations

- Fish
- Fastfetch
- Fontconfig
- GitHub CLI
- GTK 3 & GTK 4
- Kate
- Kitty
- Kvantum
- Micro
- Rofi
- Starship

## Repository Structure

```
.config/
wallpapers/
screenshots/
```

## Installation

Clone the repository:

```bash
git clone https://github.com/slsathington/dotfiles.git
```

Copy the desired configuration into `~/.config`.

GNU Stow support is planned.

## Wallpaper

The wallpaper used for this setup is included in the `wallpapers` directory.
